Google Search Engine Simulator

A micro version of Google Search Engine Simulator

- Designed and implemented a micro version of Google Search Engine GoogleSearchEngineSimulator using Java
- Allowed users to search for keywords and display lists of web URL links
- Allowed users to rank each website based on 4 factors: Frequency, Age, Money, Number of other web pages that link to the page
- Sorted lists of websites using Heapsort and stored in the Priority Queue based on ranks
- Gained experience and knowledge in Heap data structures, Heapsort, Priority Queue, and Web Crawler
